## How To

### Setup

```
git clone https://github.com/4ch1m/mozext.git
git submodule init
git submodule update
cd mozext/nestedquoteremover-me
```

### Build

```
npm install
```

### Run

```
npm start
```

### Create XPI

```
npm run package
```
